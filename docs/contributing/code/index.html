Code Contributions

A guide on how to get started contributing code to the WordPress Playground project.

Development Hub​

The WordPress Playground project uses GitHub for managing code and tracking issues. The main repository is at: https://github.com/WordPress/wordpress-playground.

Browse the issues list to find issues to work on. The good first issue labels are good starting points.

Working with the GitHub repository​

git clone https://github.com/WordPress/wordpress-playground
cd wordpress-playground
npm install
npm run dev
